ETIQUETAS HTML
ETIQUETAS DESCRIPCION
head Representa una colección de metadatos acerca del documento. Incluyendo:
  • enlaces a
  • definiciones de
  • scripts
  • hojas de estilo

El resto de etiquetas de metadatos estarán recogidas dentro de las etiquetas de apertura y cierre del head.
title Etiqueta usada para definir el título de la página web.
link Se usa para enlazar recursos externos al documento HTML.
meta Etiqueta usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo para definir el autor del sitio, o la descripción del mismo.
body Contiene todas las etiquetas de apertura y cierre que se quieran mostrar en la página web.
nav Se usa para definir el contenido que será la sección de navegación de la web.
main Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.
section Define una sección del documento
article Define contenido independiente de la web.
aside Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele estar relacionado con la web
h1, h2, h3, h4, h5, h6 Se usan para jerarquizar el contenido de la web.
header Se usan para definir la cabecera la página web. Suele contener el logotipo, menú de navegación, etc.
footer Usadas para definir el pie de página.
p Etiqueta usada para escribir párrafos de texto.
hr Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.
pre Usada para pegar texto manteniendo el pre formato propio del texto.
blockquote Se usan para indicar que el contenido es texto citado.
ol Etiquetas para crear una lista ordenada
ul Etiquetas para crear una lista des-ordenada
li Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.
dl Usada para crear una lista de definiciones.
dt Representa un término definido por la etiqueta dd
dd Se usa para definir los términos listados antes que él.
figure Indica una figura ilustrada como parte del documento HTML5.
figcaption Utilizada para definir la leyenda de una figura.
div Etiqueta común utilizada para crear un contenedor genérico.
a Etiqueta utilizada para crear hiperenlaces en el documento HTML
strong Etiqueta para definir una palabra o conjunto de ellas como importantes.
small Utilizada para dejar un comentario aparte, del tipo una nota de derechos de autoría, u otros textos que no son esenciales para la comprensión del documento.
cite Para indicar el título de una obra
sub y sup Etiquetas utilizadas para representar un subíndice o superíndice.
mark Usada para resaltar texto
span Etiqueta HTML sin ningún significado específico. Se usa conjuntamente con los atributos «class» o «id» para atribuirle ciertas características.
br Etiqueta utilizada para crear un salto de línea
img Etiqueta para «pintar» una imagen en la página web.
iframe Es una etiqueta que sirve para anidar otro documento HTML dentro del documento principal.
embed Usada para integrar una aplicación o contenido interactivo externo que no suele ser HTML.
object Utilizada llamar a un recurso externo de la web. Este recurso será tratado como una imagen, o un recurso externo para ser procesado por un plugin.
video Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.
audio Usada para cargar un archivo de audio o stream de audio.
source Permite a autores especificar recursos multimedia alternativos para las etiquetas de video o audio
svg Se usa para llamar a una imagen vectorizada.
table Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla han de ir siempre recogidas entre estas dos etiquetas.
caption Usada para indicar el título de la tabla.
colgroup Etiqueta utilizada para agrupar dos o más columnas de una tabla.
tbody Usada para describir los datos concretos de una tabla.
thead Indica el bloque de filas que describen las etiquetas de las columnas de la tabla.
tfoot Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla.
tr Se usa para indicar una fila de celdas de una tabla.
td Usada para definir una celda de una tabla.
th Etiqueta que se usa para definir el encabezado de una celda
form Etiqueta de apertura y cierre de un formulario de una página web. El resto de etiquetas de formulario deben ir siempre recogidas entre estas etiquetas de apertura y cierre de formulario.
fieldset Etiqueta que representa un conjunto o agrupación de elementos de un formulario. «Pinta» un recuadro alrededor de las etiquetas que estén contenidas dentro del fieldset
legend Etiqueta ligada a fieldset. Indica el título del fieldset
label Se usa para definir el nombre o título de un control del formulario.
input Pinta un campo de introducción de datos para el usuario. Es de las principales etiquetas de un formulario.
button Etiqueta utilizada para representar un botón en el formulario.
select Input que permite una selección entre un conjunto de opciones.
option Etiqueta ligada a select. Permite añadir diferentes opciones al select
textarea Añade un campo al usuario para que pueda introducir texto en unas líneas máximas de texto que el desarrollador puede definir.

ESTILOS DESCRIPCION SINTAXIS
Color.

Un color puede ser descrito de cualquiera de las siguientes formas:

  • Usando una palabra clave:

    se puede especificar un color usando un nombre de color predefinido

  • Usando el sistema RGB cubic-coordinate:
    • rgb(): Cada parámetro (rojo, verde y azul) define la intensidad del color entre 0 y 255.
    • rgba(): Los valores de color RGBA son una extensión de los valores de color RGB con un canal alfa, que especifica la opacidad de un color. Un valor de color RGBA se especifica con:
      rgba ( rojo, verde , azul, alfa )
      El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (nada transparente):
  • Usando el sistema hexadecimal: en la forma:
    #rrggbb _
    Donde rr (rojo), gg (verde) y bb (azul) son valores hexadecimales entre 00 y ff (igual que decimal 0-255).
  • Usando el sistema HSL cylindrical-coordinate:
    • HSL():( tono , saturación , luminosidad ) El tono es un grado en la rueda de colores de 0 a 360. 0 es rojo, 120 es verde y 240 es azul. La saturación es un valor porcentual, 0 % significa un tono de gris y 100 % es el color completo. La luminosidad también es un porcentaje, el 0 % es negro, el 50 % no es ni claro ni oscuro, el 100 % es blanco
    • hsla(): Los valores de color HSLA son una extensión de los valores de color HSL con un canal alfa,
      que especifica la opacidad de un color.
      Un valor de color HSLA se especifica con:
      hsla( tono, saturación , luminosidad, alfa )
      El parámetro alfa es un número entre 0.0 (totalmente transparente) y 1.0 (nada transparente):

  • Palabra Clave:
    style="background-color:Violet;">
  • Valor RGB:
    style="background-color:rgb(60,50,42);">
  • Valor RGBA:
    style="background-color:rgba(60,50,42,0);">
  • Valor Hexadecimal:
    style="background-color:#000000;">
  • Valor HSL
    style="background-color:hsl(10,30%,75%);">
  • Valor HSLA
    style="background-color:hsl(10,30%,75%,0);">

Backgrounds.

Las propiedades Backgrounds de CSS se utilizan para agregar efectos de fondo a los elementos:

  • CSS background-color:
    especifica el color de fondo de un elemento.
  • CSS background-image:
    especifica una imagen para usar como fondo de un elemento.
    De forma predeterminada, la imagen se repite para que cubra todo el elemento.
  • CSS background-repeat:
    Esta propiedad repite una imagen tanto horizontal como verticalmente.
  • CSS background no-repeat:
    La propiedad especifica mostrar la imagen de fondo solo una vez
  • CSS background-position:
    Se utiliza para especificar la posición de la imagen de fondo.

  • CSS background-color:

    border-style:solid;
    border-color: orange;

  • CSS background-image:
    background-image: url("img0.jpg");
  • CSS background-repeat:
    • background-repeat Horizontal:

      background-image: url("img1.jpg");>
      background-repeat: repeat-x;

    • background-repeat Vertical:

      background-image: url("img1.jpg");>
      background-repeat: repeat-y;

  • background no repeat:

    background-image: url("img1.jpg");>
    background-repeat: no-repeat;

  • background-position:

    background-image: url("img1.jpg");>
    background-repeat: no-repeat;
    background-position: left top;
Borders.

Las propiedades Borders de CSS permiten especificar: estilo, ancho y color del borde de un elemento.

  • CSS Border Style: especifica qué tipo de borde mostrar y puede tener de uno a cuatro valores (borde superior, borde derecho, el borde inferior y el borde izquierdo).
  • CSS Border Width:

    especifica el ancho de los cuatro bordes.
    El ancho se puede establecer como un tamaño específico (en px, pt, cm, em, etc.) o usando uno de los tres valores predefinidos.

  • CSS Border Color:

    Se utiliza para establecer el color de los cuatro bordes.
  • Border Style:
    • dotted- Define un borde punteado
    • groove- Define un borde ranurado 3D. El efecto depende del valor del color del borde.
    • dashed- Define un borde discontinuo
    • double- Define un borde doble
    • ridge- Define un borde estriado 3D. El efecto depende del valor del color del borde.
    • solid- Define un borde sólido
    • inset- Define un borde de inserción 3D. El efecto depende del valor del color del borde.
    • outset- Define un borde inicial 3D. El efecto depende del valor del color del borde.
    • none- No define ningún borde
    • hidden- Define un borde oculto

  • Border Width:
    • thin
    • medium
    • thick
  • Border Color:
    • nombre - especifique un nombre de color, como "azul"
    • HEX: especifique un valor HEX, como "#ff0000"
    • RGB: especifica un valor RGB, como "rgb(255,0,0)"
    • HSL: especifique un valor HSL, como "hsl(0, 100%, 50%)"
    • transparente
Margins.

se utilizan para crear espacio alrededor de los elementos, fuera de los bordes definidos.
CSS tiene propiedades para especificar el margen de cada lado de un elemento.
Todas las propiedades de margen pueden tener los siguientes valores:

  • auto: el navegador calcula el margen
  • length : especifica un margen en px, pt, cm, etc.
  • % - especifica un margen en % del ancho del elemento contenedor
  • inherit: especifica que el margen debe heredarse del elemento principal

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
Text.

  • Alineación y dirección de texto CSS:
    Especifica la dirección del texto/dirección de escritura
  • Decoración de texto CSS:

    Establece todas las propiedades de decoración de texto en una declaración

    • text-decoration-color: Especifica el color de la decoración de texto
    • text-decoration-line: Especifica el tipo de decoración de texto que se utilizará (subrayado, sobrerayado, etc.)
    • text-decoration-style: Especifica el estilo de la decoración del texto (sólido, punteado, etc.)
    • text-decoration-thickness: Especifica el grosor de la línea de decoración del texto
  • Transformación de texto:

    se utiliza para especificar letras mayúsculas y minúsculas en un texto.Se puede usar para convertir todo en letras mayúsculas o minúsculas, o poner en mayúscula la primera letra de cada palabra.

  • text-align: Especifica la alineación horizontal del texto
  • text-align-last: Especifica cómo alinear la última línea de un texto
  • unicode-bidi: Se usa junto con la propiedad direction para establecer o devolver si el texto debe anularse para admitir varios idiomas en el mismo documento
  • vertical-align:Establece la alineación vertical de un elemento
  • Transformar todo el texto en mayusculas:

    text-transform uppercase

  • Transformar todo el texto en minusculas:

    text-transform lowercase

  • Colocar letra mayuscula al inicio de cada palabra:

    text-transform capitalize

    • line-height: Especifica la altura de la línea.

    line-height: 1;

  • text-indent: Especifica la sangría de la primera línea en un bloque de texto
  • letter-spacing:Especifica el espacio entre caracteres en un texto

    letter-spacing: 2px;

  • white-space:Especifica cómo manejar el espacio en blanco dentro de un elemento.

    white-space: nowrap;

  • word-spacing: Especifica el espacio entre palabras en un texto

    word-spacing: 5px;

  • Sombra de texto: text-shadow: Esta propiedad agrega sombra al texto.

    1px 3px 2px gray, 0 0 30px rgb(3,5,30), 0 0 3px rgb(15,10,25);

Fonts. Establece todas las propiedades de la fuente en una declaración.
  • font: Establece todas las propiedades de la fuente en una declaración
  • font-family: Especifica la familia de fuentes para el texto.
  • font-size: Especifica el tamaño de fuente del texto.
  • font-style: Especifica el estilo de fuente para el texto.
  • font-variant: Especifica si un texto debe mostrarse o no en letra minúscula
  • font-weight: Especifica el peso de una fuente.
